common 패키지는 Spring Boot 프로젝트에서 공통적으로 사용되는 코드나 로직을 모아 관리하는 공간입니다. 이 패키지에는 주로 상수, 유틸리티 함수, 공통 응답 객체, 예외 처리 로직 등을 포함시킵니다. 이러한 공통 요소들을 한곳에 모아두면, 코드 중복을 줄이고 프로젝트의 일관성을 유지할 수 있으며, 나아가 유지보수성도 크게 향상됩니다.

예를 들어, API 응답에 사용되는 공통 응답 코드와 메시지를 common 패키지에 정의하면, 여러 클래스나 패키지에서 동일한 코드를 재사용할 수 있어 관리가 편리해집니다. 또한, 프로젝트에서 자주 사용하는 유틸리티 함수나 필터, 인터셉터 등을 이 패키지에 두면, 코드의 중복을 최소화하면서 공통 기능을 쉽게 활용할 수 있게 됩니다.

따라서 common 패키지는 프로젝트 전반에서 반복적으로 사용되는 로직을 한데 모아, 개발 생산성을 높이고 코드의 유지보수를 용이하게 하는 중요한 역할을 합니다.
